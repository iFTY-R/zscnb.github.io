# 介绍

计算机辅助设计(Computer Aided Design)指利用计算机及其图形设备帮助设计人员进行设计工作。

  在设计中通常要用计算机对不同方案进行大量的计算、分析和比较，以决定最优方案;各种设计信息，不论是数字的、文字的或图形的，都能存放在计算机的内存或外存里，并能快速地检索;设计人员通常用草图开始设计，将草图变为工作图的繁重工作可以交给计算机完成;由计算机自动产生的设计结果，可以快速作出图形，使设计人员及时对设计作出判断和修改;利用计算机可以进行与图形的编辑、放大、缩小、平移、复制和旋转等有关的图形数据加工工作。

# 详细描述

# 直线

在CAD界面上输入：L + 空格，选择两点连成一条直线，也可以输入多少距离的直线:💯:,也可以选择直线中点，当你的鼠标不是绿色三角形时，注意要打开F3对象捕捉→中点

注意：一条直线才输入C+空格闭合路径

# 圆

在CAD界面上输入：C + 空格，选择圆点和半径确定一个圆，也可以输入半径，直径（C +空格之后输入D）的圆 注意：打开F3对象捕捉→圆心

# 圆弧

注意：圆弧的旋转都是逆时针旋转的,注意方向

# 多段线

在CAD界面上:

  1. 利用多段线绘制圆弧：PL +空格 → 输入A ,输入R半径，切换方向按D(两点确定一个圆弧)
  2. 合并多段线：已知由多段线连成图形若干，PE + 空格 → 输入M +空格 → 框选图形并且在线段左键点击一下出现是否将圆弧等转换成多段线 → 输入y → 选择合并 → 按空格两下完成操作
  3. 有宽度的多段线： PL +空格 → 输入W(宽度)+空格 → 注意有两个点（起始宽度和终止宽度），分别输入他的宽度

# 多边形

在CAD界面上，输入POL + 空格 → 选择你需要面数 → 确定基点 → 选择内接圆还是外接圆，即可完成操作

正多边形：输入POL + 空格 → 选择你需要面数→ 点击左键一下 → 选择内接圆还是外接圆 → 输入圆的半径，即可完成操作

# 移动

在CAD界面上输入 M + 空格 → 选择对象 → 空格选择基点 → 拖动对象，操作完毕

# 复制

在CAD界面上输入CO + 空格 → 选择对象 → 空格选择基点 → 拖动复制对象，空格结束

# 拉伸

在CAD界面上输入S + 空格 → 选择对象 → 空格选择基点 → ?????

# 旋转

在CAD界面上输入RO + 空格 → 选择对象 → 空格选择基点 → 逆时针旋转多少度 → 空格结束

# 镜像

在CAD界面上输入MI + 空格 → 选择对象 → 空格选择两点镜像线 → 是否删除源对象默认不删除 → 空格结束

# 缩放

在CAD界面上输入SC + 空格 → 选择对象 → 空格选择基点 → 拖动对象缩放对象或也可以输入比例几倍

在CAD界面上输入SC + 空格 → 选择对象 → 空格选择基点 → 输入R → 再确认一个基点 → 对象缩放根据两基点连线进行缩放

# 对齐缩放

在CAD界面上输入AL + 空格 → 选择对象 → 空格选择目标点 → 空格出现端点与源点相连，有两个线段相连 → 空格出现是否基于对齐点缩放对象 → 选择是或否结束操作

# 修剪

在CAD界面上输入TR + 空格 → 选择参照物 → 空格选择裁剪对象 → 空格结束

按两下空格不需要参照物： TR+ 空格 → 空格 → 空格选择裁剪对象 → 空格结束

# 延伸

在CAD界面上输入EX + 空格 → 选择参照物 → 空格选择延伸对象 → 空格结束

按两下空格不需要参照物： EX+ 空格 → 空格 → 空格选择延伸对象 → 空格结束

# 圆角

在CAD界面上输入F + 空格 → 输入半径R空格 → 输入半径 → 选择点击两个相邻的直线

相连的直线: F+空格 → 输入半径R空格 → 输入半径为0 → 选择点击两个相邻的直线

# 倒角

在CAD界面上输入CHA + 空格 → 输入D空格 → 输入两个倒角的距离 → 分别选择相对应的直线

相连的直线：CHA + 空格 → 输入D空格 → 输入两个倒角的距离为0 → 分别选择相对应的直线

# 曲线和曲线连接

在工具栏，在绘图一栏，拟合点和控制点绘制曲线, 用光滑曲线(在工具栏的圆角一栏点开即见)实现两曲线连接

注意：拟合和控制可以相互转换，如点击目标曲线，点击倒三角转换

# 阵列

在CAD界面上输入AR + 空格 → 选择对象空格 → 选择阵列类型 → 空格结束

# CAD小技巧

# 如何输入计算公式？

快捷键：'CAL

比如：画直线：L + 空格 → 确定一点后，输入计算公式'CAL空格 → 输入计算表达式 → 空格结束

# CAD怎样等分？怎么等距？

等分： DIV + 空格 → 选择要定数等分的对象空格 → 输入要要等分几段空格→ 全选查看

等距： ME + 空格 → 选择要定距等分的对象 → 输入要要等分几段空格 → 全选查看

# 不输入命令怎么用小抓手？

点击鼠标中键实现拖动

# 彩图打印出来是黑白的？

  1. 确认你的打印机不是黑白打印机
  2. 点击大红A → 打印 → 查看选择一个打印样式表 → 点击小房子 → 特性-颜色-选择使用对象颜色 → 保存即可

# 怎样清理多余图层和数据？

框选目标对象 → W + 空格 → 目标文件保存，就可以清理多余的图层

# 块怎么编辑？怎么退出块编辑？

双击块，进入块编辑，关闭块编辑

# 十字光标怎么调整成全屏？

  1. 空白处点击右键 → 选项 → 调整十字光标大小
  2. 空白处输入OP + 空格 → 调整十字光标大小

# 填充/图案重叠，怎样调整显示顺序？

DR + 空格 → 选择目标对象 → 空格出现输入对象排序选项选择一个 → 选择参照对象 → 空格结束

# 如何快速查看图元的长度？

CTRL + 1 → 选择要查看的对象 → 拖下来就可以看到几何图形的长度

# 图能看见但却打印不出来？

图层特性 → 查看打印 → 打印机上红色的禁止符号是打印不出来了，点击它即可

# 图元选多了，怎么反选？

框选对象 → 按住键盘上shift点击不是自己想要的，即可完成操作

# 如何修改默认保存版本

OP+ 空格 → 打开和保存 → 另存为

# 如何取消选择状态？

  1. 全选：按键盘上ESC
  2. 部分选择：shift + 点击选择不选择目标
  3. 撤回： ctrl + Z

# 圆的四个顶点(象限点)不能捕捉怎么办？

点击右下方的正方形东西 → 对象捕捉设置 → 勾选象限点 → 点击确定即可

# 圆看着不圆怎么办？

重新生成圆即可

# 怎么绘制墙线、双线？

# 怎么插入特殊符号？

T+ 空格 → 查看符号@

# 怎么绘制有宽度的圆？

已知有一个圆 → PL + 空格 → w + 空格 → A + 空格 → 绘制两个圆弧 → CL + 空格

# 怎么调整CAD背景颜色？

OP + 空格 → 颜色

# 自动保存、临时保存文件在哪？

OP + 空格 → 文件
