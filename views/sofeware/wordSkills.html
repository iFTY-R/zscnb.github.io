# 3步快速搞定Word文档排版

  1. 套样式
    • 开始 → 选中文字或把鼠标定位段落后面直接选中整段 → 选择样式
    • 点击视图 ，勾选视图窗格，会把左侧出现文章大体轮廓，点击它会跳到相应位置，适用于内容多阅读不方便使用
  2. 换主题
  • 点击设计 → 点击文档格式 → 选择不同主题可实时预览
  • 如果想自定义主题的话，点击左边主题按钮更换主题
  1. 调细节
  • 点击设计 → 在右上角，点击自定义颜色 、字体、段落间距、效果，以及水印、页面颜色、页面边框等

# 驾驭Word页面设置细节

  1. 添加页眉和页码
  • 双击有直角符号那里，存在页头或页脚 → 选择样式 → 关闭页眉和页脚即可
  1. 调节页边距
  • 布局 → 页边距 → 选择默认页边距，自定义页边距
  • 视图 → 标尺 → 点击标尺上面，当鼠标变为双向箭头的话，就可以调节页边距
  1. 消除孤行（在一个页面只有一行文字）
  • 调整字体大小首先全选 ctrl + { 缩小, ctrl + } 放大
  • 调整行间距
    • 首先ctrl +A全选字体，然后在开始选项卡中，选择行和段落间距，调节大小
  • 删除文字 调节页边距
  • 孤行控制
    • 首先把光标置于孤行中
    • 然后在开始选项卡中，选择段落的下箭头，打开段落设置卡，找到换行和分页，把孤行控制勾选上即可控制

# 人人都要会的打印攻略

# 1. 如何双面打印并预留装订位？

什么时候需要双面打印？ 节省纸张，装订成册

  1. 从顶部装订，调整页边距即可，就是上面所说标尺功能即可
  2. 从侧边装订，通常双面印刷的阅读材料，为了方便翻阅，页面和页码的位置是对向的。
    • 奇偶页不同
    • 装订线 步骤：
    1. 首先把右下角的视图比例缩小
    2. 布局 → 页边距 → 自适应页边距 → 版式 → 勾选奇偶页不同
    3. 布局 → 页边距 → 自适应页边距 → 页边距 → 选择装订线大小即可

# 2.设置奇偶页不同方向的页眉页码？

奇数和偶数的页眉页脚是没有联系的，点击偶数页 页眉设置，页码右对齐即可

# 3. 缩放打印更省纸

文件 → 打印 → 选择最后每版打印2页 → Microsoft Print to PDF → 打印按钮

# 4.打印格式错乱

保存PDF格式即可

# 设定模板样式

对于大型文档，需要学会统筹规划，才能事半功倍

  1. 设计范本

    • 梳理内容层级
      • 先在纸上梳理内容层级，初步确认需要多少个级别的标题
    • 设置标题格式
      • 修改标题格式
        • 利用内置的标题样式，给内容快速分级显示
        • 需要修改标题格式时，直接修改样式就能批量更新格式。
    • 添加多级编号
      • 利用多级列表，可以给各级标题快速添加自动编号，而且自动关联、自动延续

  2. 套用模板

    • 设置快捷键
      • 标题栏点击右键，然后修改，然后左下角看到格式按钮，点击快捷键进去，输入新的快捷键，保存如果选择第一个，所有word都使用，第二个仅使用于此文档。点击指定，如果看到出现在当前快捷键里面时，就成功了，点击关闭即可
    • 存成模板反复用
      • 使用模板规范所有的样式
        • 进入要保存的模板 → 文件 → 另存为 → 浏览 → 把保存类型选择word模板即可
        • 如何导入呢？ 点击另一个word文档，点击开始按钮，点击第三个样式的箭头，点击最后面的 管理样式，然后呢点击左下角的导入/导出,然后关闭文件，然后打开文件，找到自己保持word模板路径，按shift键选择要复制标题样式，点击复制，
    • 多人分工合作
      • 把制作好的Word模板分发给别人
      • 组织一次培训，把使用方法告诉大家
      • 整理一个计划表，让自己跟进大家的进度

  3. 批量合并

    • 解决分工合作
      • 编写内容时，要确定好分工内容和命名规则，使用标题都成一个文档
    • 快速合并文档
      • 插入 → 选择最右边的对象功能，点击文件中文字，把要合并的文档插入
    • 快速调整格式
      • 利用样式更新匹配功能，我们可以实现全文修改格式

# 批量打印工资条

邮件合并的原理： 实际上邮件合并的功能就是我们正在编辑word文档作为一个模板对应Excel每一条记录都会复制这个模板，然后把Excel数据填入这个模板中，所有我们看到每一页都一条工资条

如何在每一页显示多条工资条呢？

  1. 复制工资条

  2. 点击左下角，有箭头那种，点击规则，选择下一条记录，最后一条工资条不需要

  3. 点击完成并合并，点击单个文档，勾选全部点击确定

  4. Art + F9 键显示域名代码，#0.00 代表只保留两位小数 ,再次按Art+ F9关闭域名代码

要点：

  1. 工资条模板和数据

纸张方向横向，页边距要窄，可以显示更多数据。

  1. 预留裁剪区域

合并打印需要记住回车空一行，预留裁剪区域

  1. 下一条记录

利用规则中下一条记录，可以在同一页中排更多数据记录
