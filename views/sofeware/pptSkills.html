PPT你应该知道20个小技巧

vuePress-theme-reco Mr.Zhao    2019 - 2020

PPT你应该知道20个小技巧

Mr.Zhao 2020-03-05 19:47:54 重点

# 提取素材

  1. 首先你把需要ppt复制一份保存，在副本上操作
  2. 把副本重命名，并更改副本文件扩展名，如zip格式，点击解压当前文件夹
  3. 打开解压文件，然后点击ppt文件夹，进入media文件夹，就得到我们所需的素材了

# 压缩

# 压缩媒体

选中视频 然后点击 文件 → 信息 → 压缩媒体 → 根据所需空间大小选择要压缩的尺寸

# 压缩音频

选中音频 然后点击 文件 → 信息 → 优化媒体兼容性 → 根据所需空间大小选择要压缩的尺寸

# 压缩图片、

选中双击图片 出来一个压缩图片 然后点击进入，如果只压缩此张图片 ，打勾仅应用于本张图片；如果ppt上所有图片，把勾去掉即可，选择合适分辨率

# 编号

# 加项目编号

坑：在一个文本框写文字

Alt + Shift + ↑ 可以选中文字向上移动

Alt + Shift + ↓ 可以选中文字向下移动

# 有序编号：

选中文字 → 点击右键 → 可以选择有序编号，

# 项目编号(无序编号)

选中文字 → 点击右键 → 可以选择项目编号，图片，自定义符号等

# 图表个性化

点击一次选中一个系列，再点击一次选中单个元素，

可以加图片首先剪切到剪贴板里，选中条形图，再复制一下，图片就粘贴进来了，此时我们看图片是伸展的，

选中条形图，右键选择数据格式，来到填充，把伸展改为层叠即可

# 图表动画

在动画窗格那里，双击对应所有动画名称，就出现一个窗口，选择图表动画 → 按分类

如果我们不希望背影有动画，在动画窗格删除即可

# 转化SmartArt

选中文字 → 右键 -- 转换成转化SmartArt ，就可以看到不同样式，都带实时预览的

选中所有图片，就出现了图片样式，点击带实时预览的各种样式，如果我们想删除文本框，直接删除会把图片也一样删除点，怎么办？

大招： 右键 → 组合 → 取消组合 就可以格式化图片了

# SmartArt动画

进入动画选项卡 -- > 选择效果选项 → 打开动画窗格 → 双击所有图示 → 选择SmartArt动画 → 逐个

# gif 和png图片

百度上搜索关键字后面加上gif就可以搜索到动图

# 取色器

形状，文字，背景都能取色

如果我们想要文件名标题栏的颜色，是取不到呢，怎么办？

一直按住左键到指定区域，然后松开，背景就跟文件名背景一样，是不是很神奇！！

# 布尔运算

插入 → 一个圆形 → 按住shift画出正圆，选中正圆然后ctrl + D复制一份

然后按住ctrl键选中两个圆，然后形式格式看到合并形状，选择拆分，然后吸取颜色，最后选择所有图片，在形状轮廓那里选择无轮廓，就搞定了

# 嵌入字体

文件 → 选项 → 保存 → 将字体嵌入文件 →嵌入所有字符

，这样一来，别人也能自己的电脑看到你字体样式，是不是很神奇呢！

# 粘贴为图片

只使用几个字，还要嵌入字体，很占容量，该怎么办呢？

选中文字，先剪切后按复制一下，就可以右下角出现ctrl，点击它选择图片

# 抠图

# 复杂图片

1 . 点击头像图片 → 左上角删除背景，背影就变成紫色 2. 点击+号选择要保留的区域 ，然后点击保留更改，背景变成透明的图片了！！！

# 简单图片（只要一种背景色时）

选择拥有白色背景的图片。选中图片，在颜色选中透明色，鼠标变成下箭头时点击白色区域，，背景变成透明的图片了！！

# 图片裁剪

双击图片 → 找到裁剪 → 纵横比 → 调节图片要裁剪的部分 → 在图片外单击就可以完成裁剪

双击图片 → 找到裁剪 → 裁剪为形状,

如果快速裁剪圆形呢？

双击图片 → 找到裁剪 → 裁剪为形状 → 点击圆形 → 在纵横比为1:1，就完美得到圆形了。

# 增减水印

视图 → 幻灯片母版 → 要找到最上面一张，增减水印就可以了，最后关闭母版视图

# 格式刷 & 动画刷

# 格式刷（一些静止）

选中图片 → 单击格式刷 → 在目标图片上轻轻地刷一下即可

选中文字 → 双击格式刷 → 在目标文字轻轻地刷一下即可，可以跨屏刷，取消在文字外面单击一下即可

# 动画刷（一些有动画的图片）

动画 → 选中图片 → 单击动画刷 → 在目标图片上轻轻刷一下即可，自动取消

选中文字 → 双击动画刷 → 在目标文字轻轻地刷一下即可，可以跨屏刷，取消在文字外面单击一下即可

# 导出PDF格式

优点：

  • 内容格式不会变型
  • 手机方便查看

文件 → 导出 → 创建PDF/XPS → 选择保存位置 → 点击确定

# 插入3D模型

插入 → 3D模型 → 选择要插入3D模型

# Prezi效果 幻灯片缩放定位

坑： 一定要再次击缩放定位，一直出现下图即可

插入 → 缩放定位 → 幻灯片缩放定位 → 选择所需幻灯片 → 再次点击缩放定位 → 缩放定位背景去除即可 → 勾选上返回缩放定位即可

# 平滑切换

图片，文字，图形都使用

选中整个幻灯片 → ctrl + D 复制一份 → 在副本调节其位置 → 在切换里选择平滑，在效果选项里选择相应的形式
